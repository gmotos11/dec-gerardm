1. Què és Vanilla Javascript?

Vanilla Javascript és com es coneix al llenguatge JavaScript quan s'utilitza sense cap llibreria o framework. En un inici era l'única manera d'utilitzar JavaScript a les pàgines web, però poc a poc s'ha anat complicant

2. Diferència els ECMA Script que pertanyen a Javascript i a Javascript Modern.

Els 2 són llenguatges de programació, la diferencia és que JavaScript també implementa ECMASScript, O sigui que JS es beneficia d'aquest altre.

3. Qui va crear Javascript? De quina empresa era?

Es nececitaba un llenguatge de programació que s'executés en el navegador de l'usuari. Brendan Eich, un programador que treballava a Netscape, va pensar que podria solucionar aquest problema adaptant altres tecnologies existents. Eich va denominar al seu llenguatge LiveScript. La raó de el canvi de nom va ser exclusivament per màrqueting, ja que Java era la paraula de moda en el món informàtic i d'Internet de l'època.

4. Quina empresa hi ha actualment darrera de Javascript?

JavaScript es troba oficialment sota l'organització de Mozilla Foundation, i periòdicament s'afegeixen noves característiques de l'llenguatge.

5. Què és una arrow function? A quin ES apareix (quin any).

Són una nova forma de definir funcions i hi ha diferents variants en la sintaxi
Al crear una arrow function d'un sol paràmetre no cal escriure els parèntesis, tampoc cal escriure les claus, això es pot quan la funció és d'una sola línia i retorna un valor.
